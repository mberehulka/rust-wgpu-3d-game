## Rust 3d game with [WGPU](https://wgpu.rs/)

- Todo:

- Running

    - Compile animations:
    
            > blender --background --python compiler.py
        
    - Compile meshes and textures:
    
            > cargo run --bin compile --release
            
    - Running:
    
            > cargo run --release
